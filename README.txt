# De Dignis Insanis Opera — Site (HTML + CSS + JS)

## Como usar
1) Abra o arquivo `index.html` no navegador.

## Colocar suas imagens (mantendo nomes)
Coloque os arquivos abaixo exatamente nesses caminhos:

- Fundo (plano de fundo do site):
  - `assets/background.jpg`

- Personagens (cards):
  - `assets/characters/terina.jpg`
  - `assets/characters/topazio.jpg`
  - `assets/characters/kimiona.jpg`
  - `assets/characters/shay.jpg`
  - `assets/characters/shinno.jpg`
  - `assets/characters/lucio.jpg`
  - `assets/characters/mestre.jpg`

> Se seus nomes de arquivos forem diferentes, abra `script.js` e altere o campo `img:` de cada personagem.

## O que foi melhorado
- Fundo com opção de ficar **fixo** ou **não fixo** (toggle no topo)
- Animações suaves ao rolar (você pode desligar)
- Cards de personagens gerados via JS + busca + filtros
- Hover com brilho e tilt (opcional via toggle de animações)
- Modal com detalhes do personagem (pronto para você linkar para páginas separadas)
